# 💍 Malshi & Sadeep — Digital Wedding Invitation

A premium, Sri Lankan-inspired digital wedding invitation with RSVP system and admin dashboard.

**10th September · Lighthouse Hotel, Galle, Sri Lanka**

## Quick Start

1. Customize content in `public/index.html` (time, names, contacts)
2. Change admin password in `api/get-rsvps.js`, `api/export-csv.js`, and `public/admin.js`
3. Push to GitHub → Deploy on Vercel

## Admin Panel
URL: `/admin.html` · Default password: `wedding2025`

## Files
- `public/index.html` — Main invitation page
- `public/admin.html` — Admin dashboard
- `api/rsvp.js` — Save RSVP submissions
- `api/get-rsvps.js` — Fetch all RSVPs
- `api/export-csv.js` — Export CSV

See the full deployment guide for detailed instructions.
